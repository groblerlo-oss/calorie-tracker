1. 👟 STEP COUNTER INTEGRATION
================================================================

WHAT IT DOES:
- Track your daily steps
- Automatically calculates calories burned
- Formula: 1,000 steps ≈ 40 calories burned
- Reduces your net calorie intake

WHERE TO FIND IT:
- Food Tracker tab
- Orange section between Exercise Logger and Add Food

HOW TO USE:
1. Enter your daily steps (e.g., 10,000)
2. Click "Log Steps"
3. Steps convert to calories burned automatically
4. Net calories = Food - Exercise - Steps

EXAMPLE:
- Food: 2,000 cal
- Exercise: -300 cal
- Steps (10,000): -400 cal
- NET: 1,300 cal ✓

BENEFITS:
- Rewards daily activity
- Encourages more walking
- More accurate calorie tracking
- Earn extra food budget!

FIREBASE SETUP (IF NEEDED)
================================================================

NO CHANGES NEEDED if already using Firebase!

The app automatically creates:
- Collection: "admin_tracking"
- Documents contain:
  {
    event: "user_registered" or "user_login",
    email: "user@email.com",
    timestamp: "2026-03-06T16:45:00.000Z",
    userId: "firebase_user_id"
  }

FIRESTORE RULES (ensure these are set):
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId} {
      allow read, write: if request.auth != null
                        && request.auth.uid == userId;
    }
    match /admin_tracking/{docId} {
      allow read: if request.auth != null;
      allow write: if request.auth != null;
    }
  }
}
